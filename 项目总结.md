### 3. 后端架构实现

#### Controller 层
```java
MockInterviewController.java
- GET  /                              首页
- GET  /upload                        上传页面
- POST /api/resume/upload             上传简历并评分
- GET  /analysis/{resumeId}           分析结果页面
- GET  /api/resume/{resumeId}/analysis 获取分析详情
- GET  /interview/{resumeId}          面试页面
- POST /api/interview/{resumeId}/questions 生成面试问题
- POST /api/interview/{resumeId}/submit   提交答案
- GET  /result/{resumeId}             评估结果页面
```

#### Service 层
```java
MockInterviewService.java
- scoreResume()           评分简历
- generateInterviewQuestions() 生成面试问题
- evaluateAnswers()       评估答案
- saveResume()            保存简历数据
- saveQuestions()         保存问题
- saveEvaluation()        保存评估结果
- getResumeById()         获取简历数据
```

#### DTO 类
```java
ResumeScoreResult.java      简历评分结果
InterviewQuestions.java     面试问题
InterviewEvaluation.java    面试评估
ResumeData.java            简历完整数据
```

## ⚠️ 已知限制

1. **文件格式**: 目前仅支持文本格式读取（PDF/Word 需要额外解析库）
2. **数据持久化**: 使用内存存储，重启后数据丢失
3. **BrowserTool**: 已暂时禁用（避免 ChromeDriver 依赖问题）
4. **用户系统**: 暂无用户登录和历史记录功能
